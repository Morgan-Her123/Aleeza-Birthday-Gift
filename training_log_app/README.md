# Training Log App

A standalone training log web app inspired by TrainingPeaks-style workflows.

## Features

- Dashboard with weekly load, rolling volume, and recent workouts
- Calendar/log view grouped by week
- Workout detail pages
- Manual workout entry
- SQLite-backed storage with seeded sample data

## Run

```bash
cd "/Users/minghe/Documents/New project/training_log_app"
python3 server.py
```

Then open the local URL printed in the terminal.
